# Парсер объявлений с otomoto.pl

Скрипт для сбора ссылок на объявления о продаже автомобилей с польского сайта otomoto.pl.

## Назначение

Сбор структурированных данных с сайта объявлений для дальнейшего анализа рынка автомобилей.

## Функциональность

- Сбор ссылок на объявления с главной страницы раздела "Легковые автомобили"
- Настройка HTTP-сессии с автоматической подменой User-Agent

## Используемые технологии

- Python 3
- Requests
- BeautifulSoup4
- fake-useragent

## Структура

- `parse_otomoto.py` — основной скрипт с функциями сбора и сохранения данных

## Запуск

Установка зависимостей:
pip install requests beautifulsoup4 fake-useragent
